# Initialization

There are several ways to go about `Initialization` within C++.
``` cpp linenums="1"
int main()
{
	//Copy Initialization
	int a = 5;
	//Direct Initialization
	int b(5);
	//List Initialization
	int c{5};
}
```

`List Initialization` is preferred between all the above methods.
